Code Forces 1573 Problem F

Figure Fixing

June 28, 2021

Problem statement: Figure Fixing

Analysis

This problem is not difficult once some observations are made.

  1. The sum of the target values must have the same parity as that of the initial values.
    2.  This is because the values are updated 2 at a time, ie the parity remains the same.
  2. If  (1) is satisfied, and the graph is not bipartite, there always exists a solution.
    3.  This is because we can cyclically update the values. Color the graph bipartite, with some number of adjacent colors. Then we can make the sums of the values equal.
  3. If  (1) is satisfied, and the graph is bipartite, then a solution exists if and only if the sum of the differences between the target and the initial values of each parity is the same.
    4.

Time Complexity: $\mathcal{O}(N+M)$

Reason: BFS to check if the graph is bipartite
